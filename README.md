## ⚙️ How to Install Mods?

📂 **1. Open Dota 2 game folder**  
Go to your Dota 2 installation directory.

📂 **2. Create required folder**  
Inside the `game` directory, create a new folder:
- `dota_123` — for English version of the game  
- `dota_russian` (or other) — if your game uses another language  

---

🔄 **3. Install the mod**  
Move the `.vpk` file into the created folder (`dota_123` or equivalent).

---

⚠️ **Important Note**  
Some mods may have different installation methods.  
For example, cursor or terrain mods may include folders like `mods` or `cursors` inside the archive.  
Always read the instructions included in the mod package.

---

🚩 **Final Step**  
Add this to your Dota 2 launch options:
-language 123

If your game is not in English, use:
-language russian
(or your language code)
